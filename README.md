# Rock-Paper-Scissors
This project simulates a Rock–Paper–Scissors game where one player follows a probabilistic strategy modeled as a Markov chain. The second player learns this strategy using Bayesian updating (Dirichlet prior) and predicts the opponent’s next move to select an optimal counter-move.
